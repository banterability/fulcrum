fulcrum
=======

[Pivotal Tracker](http://www.pivotaltracker.com/) status at a glance. Designed for phones, TVs, and low-bandwidth video conferences.


Getting Started
---------------

Install node modules:

```bash
npm install
```

Visit `make` for usage instructions.
